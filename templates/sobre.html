Sobre o Projeto

Este é um mini app em Flask que integra a API do The Movie Database (TMDB) para recomendar filmes similares a um título informado pelo usuário.

  • Rotas: / (busca e resultados) e /sobre (esta página).
  • Funcionalidades: busca de ID do filme, recomendações, paginação simples e cards com poster/ano/sinopse.
  • Boas práticas: uso de python-dotenv e variáveis de ambiente para proteger a API Key.

Para rodar localmente, siga o README no repositório.
